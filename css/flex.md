
فرض کن به پرنت کلاس فلکس دادی و حالا چندتا ستون داریم اما چون محتوای ستونها با هم فرق دارن این ستون های اندازه/عرض یکسانی ندارن... برای اینکه عرض ستون ها فارغ از اینکه محتواشون چقدره یکسان باشه میایم و به کل چیلدرن های اون پرنتی که بهش فلکس دادیم میایم و فلکس بیسیس صد درصد میدیم
```css
.parent > * {
  flex-basis: 100%;
}

```


```css
.parent {
  display: flex;
  flex-wrap: wrap;

  /* تا اینجای کار اگر 3 تا کارت داشته باشیم تو سایز تبلت سومی میره خط بعدی ولی کل خط رو نمیگیره... حالا برای اینکه سومین کارت که رفته خط بعدی کل خط رو بگیره بهش فلکس گرو میدیم تا بتونه گرو کنه و کل خط بعدی رو بگیره و زشت نشه */
  flex-grow: 1;
  /* یکی دیگه از مزایای فلکس گرو و اینکه کجا بدرد میخوره برای وقتیه که فلکس رپ دادی بعد چون یکی از کارت ها رفته صفحه بعدی اون گوشه یکم فضای خالی افتاده... حالا اگه فلکس گرو بدی اون اندازه کارت ها به قدری بزرگ میشن که اون گوشه پر بشه */


  /* حالا ممکنه بگی که خب الان سومین کارت خیلی بزرگ شد... در اینصورت دیگه باید اسکرول افقی بدی بهش : */
  overflow-x: scroll;
}
```


```css
/* اما درنهایت بهترین کار برای ریسپانسیو کردن استفاده از گرید هست */

  display: grid;

  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
```

```css
.parent {
  display: flex
}
.child {
  /* مقدار فلکس از سه مقدار تشکیل شده */

  /* flex-shrink */
  /* flex-basis */
  /* یا میتونیم برای فلکس سه تا عدد بذاریم که به ترتیب نشون دهنده اینا باشه یا اینکه اگه برای فلکس عدد 1 رو بذاریم ینی مقدار گرو و شرینک برابر با 1 و مقدار بیسیس برابر با 0 هست  */
  /* که میشه به این سه حالت نوشتش: */
  /* flex: 1; */

  /* flex: 1 1 0; */
  
  /* flex-grow: 1; */
```
