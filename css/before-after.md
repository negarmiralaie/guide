وقت هایی که استایل دایره های بولت پوینت های لیست ها رو دوست نداری باید از after استفاده کنی
```css
h1 {
  position: relative;
  width: max-content;
}

/* وقتی این بیفور رو مینویسی مثل اینه که انگار داری یه المنت خالی قبل از تک اچ1 درست میکنی */
/* حالا چون یه المنت خالیه باید همه چیزش رو بهش بدی حتی کانتنتش رو */
h1::before {
  content: 'Hello';
  content: '*';
  /* اگه ازین المنت جدید فقط برای دیزاین میخوای استفاده کنی و قرار نیست محتوایی داشته باشه باید کانتنت رو خالی بذاری البته حتمااا باید کانتنت رو بنویسی تا کار کنه فقط باید خالی باشه */
  content: '';
  /* حالا اگه کانتنت رو خالی بذاری میبینی که هیچی در صفحه نشون داده نمیشه حتی اگه عرض و ارتفاع بهش بدی... چون باید پوزیشن ابسلوت هم بهش بدی و به پرنتش هم پوزیشن رلتیو بدی تا بیاد داخل صفحه و بتونی ببینیش */
  position: absolute;
  bottom: 0;
  left: 0;
  height: 2px;
  /* حالا اگه خواستی یه خط کشیده بشه زیر اچ1 باید به این بیفور یا افتر یه عرض 100% بدی تا کلش رو بگیره البته چون اچ1 بلاک المنته تا اخر خط رو میگیره و باید به اچ1 مکس ویت بدی یا دیسپلیش رو عوض کنی */
  width: 100%;

/* برای اینکه دیده بشه باید بهش رنگ بکگراند بدی */
  backgroud: red;

  /* میتونی رنگ بکگراندش رو گردینت بدی تا عین یه خط گردینت دار بشه */
  background: linear-gradient(to right, red, blue);

  /* میتونی بهش بردر ردیوس هم بدی */
  border-radius: 30px;

  transition: 100ms;
}

h1:hover::after {
  /* اینطوری انیمیشن هم داره که وقتی روی اچ1 هاور کردی از بین بره */
  width: 0;
}
```
