ما هم به متن میتونیم شدو بدیم و هم به یک المنت

```css
h1 {
  text-shadow: 10px 50px 5px red;
  /* یه کپی از این متن ساخته میشه که رقم اول نشون میده پوزیشن افقی این متن تکراری کجا باشه و رقم دوم پوزیشن عمودی رو نشون میده و با رقم سوم میتونیم میزان بلر بودن اون رو مشخص کنیم بعدشم میتونیم رنگش رو مشخص کنیم */ 

  text-shadow: 10px 50px 5px red, -10px -50px 5px blue;
  /* میتونیم چندتا تکست شدو بدیم و با کاما از هم جداشون کنیم... همشون اعمال میشن */

  box-shadow: 10px 50px 5px red;
  /* از تکست شدو برای متن ها و از باکس شدو برای شدو دادن به بقیه المنت ها استفاده میکنیم */
}
```
