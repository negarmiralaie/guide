```css
.element {
  /* این مقدار دیفالته که هیچ تاثیری نداره کلا */
  position: static;

  /* درصورتی که پوزیشن المنت رو برابر با یکی از 4 مقادیر زیر قرار بدی میتونی از top bottom right left استفاده کنی */
  /* absolute */
  /* relative */
  /* sticky */
  /* fixed */

}
```


* position: relative:
باعث میشه بتونی با تاپ و باتم و .... المنت رو نسبت به جایی که همون اول بوده ینی وقتی که پوزیشن بهش نداده بودی و پوزیشنش استتیک بوده تغییر کنه 
```css
.element {
  position: relative;
  /* حالا که اینا رو بهش بدی از جای اولیشه از سمت بالا 50پیکسل میاد پایین و از سمت چپ 20 پیکسل میاد راستتر */
  top: 50px;
  left: 20px;
}
```


* position: fixed
باعث میشه موقعیت المنت ما تغییر نکنه و اگه سایتمون اسکرول عمودی داشته باشه ما هرچی اسکرول کنیم سمت پایین  این از جاش تکون نمیخوره
حالا تاپ و رایت و اینا نسبت به ویندو محاسبه میشن
اینطوری میشه این تبلیغاتی که روی صفحه سایت میمونن و تکون نمیخورن یا این مدال های اکسپت کوکی رو درست کرد

```css
.element {
  position: fixed;
  top: 50%;
  left: 50%;
  /* تا اینجا فقط گوشه های باکس ما اومده وسط صفحه */
  /* ولی ما میخوایم دقیقا مرکز باکس ما بیاد وسط صفحه پس با استفاده از ترنسلیت میایم و باکس و به نسبت خودش 50% جابجا میکنیم تا کاملا بیاد وسط */
  translate: -50% -50%;
}
```


* position: sticky;
تفاوتش با پوزیشن فیکسد اینه که همیشه فیکسد نیست... ما میایم با تاپ یا باتم یه عددی بهش میدیم حالا المنت تا قبل ازینکه به اون نقطه ای که با تاپ و باتم براش مشخص کردیم برسه مثل بقیه تگ ها عمل میکنه و وقتی به اون نقطه از صفحه برسیم هنگام اسکرول کردن دیگه فیکسد میمونه روی صفحه و تکون نمیخوره


* position: absolute;
برای وقتایی که میخوایم موقعیت المنت رو به نسبت یه المنت دیگه تغییر بدیم... مثلا بره داخل یه المنت دیگه
مثالش ایکون سرچ داخل اینپوت سرچ هست که دقیقا داخلش قرار گرفته
یا آیکون چشم که در اینپوت پسورد قرار میگیره

نکتش اینه که حتما باید پرنت مدنظرش پوزیشن رلتیو داشته باشه تا نسبت به اون تنظیم بشه... پوزیشن رلتیو تاثیر خیلی زیادی روی ظاهر المنت پرنتش نمیذاره و فقط برای این بچش که قراره ابسلوت بشه تاثیر داره... حالا که به پرنت رلتیو و به بچه ابسلوت دادیم دیگه با تاپ و اینا میتونیم موقعیتش نسبت به پرنتش رو تغییر بدیم


اگه هیچکدوم از پرنت هاش پوزیشن رلتیو نداشته باشن به نسبت تگ بادی تنظیم میشه
